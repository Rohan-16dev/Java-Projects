# Java Development Internship Tasks

This repository contains the Java programs I implemented during my **Java Development Internship at Cognifyz Technologies**.

The internship focused on strengthening core Java concepts through a series of logical programming tasks covering topics such as input/output handling, string manipulation, arrays, file handling, and problem solving.

---

## Internship Details

- **Role:** Java Development Intern  
- **Organization:** Cognifyz Technologies  
- **Duration:** November 2025 – December 2025  
- **Domain:** Core Java Programming

---

## Tasks Completed

### Level 1

1. **Temperature Converter**
   - Converts temperature between Celsius and Fahrenheit based on user input.

2. **Palindrome Checker**
   - Checks whether a given word or phrase is a palindrome while ignoring spaces and punctuation.

3. **Student Grade Calculator**
   - Accepts multiple grades from the user and calculates the average grade.

4. **Random Password Generator**
   - Generates a random password based on user-defined length and character types (uppercase, lowercase, numbers, special characters).

---

### Level 2

5. **Tic-Tac-Toe Game**
   - A two-player console-based Tic-Tac-Toe game using arrays and conditional logic.

6. **Password Strength Checker**
   - Evaluates the strength of a password based on length and character composition.

7. **File Encryption / Decryption**
   - Encrypts and decrypts text files using a basic encryption technique and Java file handling.
